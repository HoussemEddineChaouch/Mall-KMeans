# Mall Customer Segmentation K-Means

## Description
This project applies K-Means clustering to segment mall customers based on:
- Age
- Annual Income

## Features
- Data exploration and cleaning
- Visualization of clusters
- Elbow method to find optimal K
- Final clustering with visualization

## Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Results
Customers are grouped into clusters based on similarities in age and income.
